# Queendom Survivors

A survivor-style action game made with Godot 4.

## Getting Started

1. Clone the repo
2. Open the project in Godot 4
3. Hit **F5** to run

## Project Structure

```
queendom-survivors/
├── player.gd
├── enemy.gd
├── bubblegum_ball.gd
├── world.gd
└── icon.svg
```

## Built With

- [Godot 4](https://godotengine.org/)
